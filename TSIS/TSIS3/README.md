# TSIS 3 — Racer Game

## How to run
```
pip install pygame
python main.py
```

## Controls
- Arrow keys — move car
- ESC — return to main menu

## Features
- Main Menu / Settings / Leaderboard / Game Over screens
- 3 difficulty levels (Easy / Normal / Hard)
- Power-ups: Nitro (speed boost), Shield (one-hit protection), Repair (clears hazards)
- Lane hazards: oil spills, potholes, slow zones, speed bumps
- Road events: nitro strip boosts
- Multiple enemy traffic cars with difficulty scaling
- Persistent leaderboard (top 10) saved to leaderboard.json
- Settings (sound, car color, difficulty) saved to settings.json
- Username entry before play
